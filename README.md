# Exercícios de Sistemas Digitais 
Este diretório contém as resoluções da primeira lista de exercícios da disciplina de Sistemas Digitais.

## Informações da Disciplina

* **Universidade:** Universidade Federal do Ceará (UFC) - Campus Quixadá 
* **Disciplina:** Sistemas Digitais.

## Fonte dos Exercícios

Todas as questões referem-se ao livro:

* **Título:** Sistemas Digitais, 2ed.
* **Autor:** Frank Vahid

## Simulações

Para validar e visualizar o comportamento dos circuitos, algumas das questões foram implementadas e testadas usando um simulador lógico digital.

* **Software Utilizado:** Digital - https://github.com/hneemann/Digital
* Os arquivos de simulação estão linkados diretamente nas resoluções.
